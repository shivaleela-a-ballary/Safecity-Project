# 🚨 Safe City – Women Safety Platform

**Developed as part of an academic project to explore how technology and data can contribute to improving women's safety in urban environments.**

Safe City is a women’s safety web app providing **location-based safety insights** and **awareness resources** for safer urban navigation.

## 🌟 Features

- **🤖 AI Risk Analysis** - Location-based risk detection & alerts
- **📍 Real-time Safety Insights** - Nearby area safety scores
- **🔐 Firebase Auth** - Secure user login
- **📚 Awareness Resources** - Self-defense, legal rights, emergency tips
- **🎨 Clean UI** - Simple, intuitive design

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| **Frontend** | HTML, CSS, JavaScript |
| **Backend** | Firebase |
| **Auth** | Firebase Authentication |
| **Data** | Location APIs, Safety Analysis |
